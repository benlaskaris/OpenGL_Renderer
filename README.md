# OpengGL

Basic Renderer using OpenGL, GLEW and GLFW. Designed using the concept of a scenemanager, and drawable obejcts. Built and tested on OSX. Not yet cross-platform.

# Libraries/Tools needed

-cmake
-glew
-glfw
-glm


# To Build

1. clone the directory
2. mkdir build
3. cd build
4. cmake .. 
5. make
6. ./test
